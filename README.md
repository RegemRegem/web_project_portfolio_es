# Portafolio de Erick Baltazar Escobar

Portafolio profesional de Erick Baltazar Escobar, desarrollador Full-Stack Junior. El sitio presenta información personal, habilidades técnicas, proyectos destacados y medios de contacto.

## Funcionalidad

- Presentación profesional.
- Sección de habilidades y tecnologías.
- Galería de proyectos destacados.
- Navegación interna con desplazamiento suave.
- Diseño adaptable a computadoras, tabletas y dispositivos móviles.

## Tecnologías y técnicas

El proyecto utiliza HTML5 semántico, CSS3, metodología BEM, Flexbox, Grid, consultas de medios y JavaScript para el desplazamiento suave. Las fuentes Open Sans y Archivo Black se alojan localmente.
